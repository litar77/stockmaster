# 模块2：进攻策略（牛市 + 热市）

## 策略定位

市场处于牛市且交易活跃，是赚取超额收益的最佳时机。核心思路：**寻找龙头爆发，快进快出，保住盈利**。

## 选股条件

### 主要筛选标准

从全市场约5000只股票中，按以下条件逐步筛选：

**第一轮：趋势过滤**
- 均线多头排列：MA5 > MA10 > MA20 > MA60
- 股价站上MA20，且MA20向上
- 近5日涨幅 > 5%（排除横盘无趋势股）

**第二轮：量能确认**
- 近5日日均成交量 > 20日均量 × 1.5（放量确认）
- 换手率在 3% ~ 15%（活跃但不过度投机）
- 量比 > 1.2（当日量能积极）

**第三轮：强势特征**
- 近10日内有涨停板记录（龙头特征）
- 所属行业板块近5日涨幅排名前20%（板块共振）
- 相对大盘超额收益 > 3%（强于大盘）

**第四轮：缠论确认**
- 日线级别出现第一类买点或第二类买点
  - 第一类买点：下跌趋势中，最后一个中枢的第三类卖点后出现底分型
  - 第二类买点：第一类买点后第一次回调不破前低
- 30分钟级别至少有一个向上的中枢形成

### 排除条件

- ST、*ST 股票
- 上市不满60个交易日的新股
- 停牌或涨跌停无法交易的股票
- 市值 < 30亿（流动性不足）

## 买入信号

满足以下任一条件触发买入：

| 信号 | 条件 | 确认方式 |
|------|------|----------|
| 放量突破 | 股价突破近20日最高价，成交量放大1.5倍以上 | K线收盘价确认 |
| 涨停次日 | 涨停次日高开（>2%）且不回补缺口 | 开盘30分钟内确认 |
| 缠论买点 | 出现第一类或第二类买点 | 底分型成立确认 |
| 强势回踩 | 连续上涨后回调至MA5附近获支撑，缩量企稳 | 缩量+收阳线确认 |

## 卖出信号

| 信号 | 条件 | 操作 |
|------|------|------|
| 目标止盈 | 盈利达到 15%-20% | 减仓50%，剩余用移动止盈 |
| 移动止盈 | 从最高点回落超过 5% | 清仓 |
| 破位止损 | 跌破MA5且次日不能收回 | 清仓 |
| 固定止损 | 亏损达到 -5% | 无条件清仓 |
| 缠论卖点 | 出现第一类或第二类卖点 | 至少减仓50% |
| 量价背离 | 价格创新高但成交量萎缩 | 减仓30%，观察 |

## 仓位管理

| 参数 | 设置 |
|------|------|
| 总仓位上限 | 80% |
| 单票仓位上限 | 20% |
| 同时持股数量 | 3-5只 |
| 单次建仓比例 | 先买入目标仓位的50%，确认后加满 |
| 每日最大操作次数 | 不限，但需每笔有明确理由 |

## 操作节奏

- **开盘（9:15-9:30）**：观察竞价情况，关注高开超3%的候选股
- **早盘（9:30-10:30）**：最佳买入时段，龙头股通常在此期间确立
- **盘中（10:30-14:00）**：观察为主，关注缩量回踩机会
- **尾盘（14:00-15:00）**：评估全天走势，决定是否加仓或减仓

## 持仓操作指令规则

当策略为"进攻"时，对持仓股的操作判断：

- **加仓**：盈利中且回踩MA5/MA10获支撑 + 量能未萎缩 → 加仓至目标仓位上限
- **持仓**：趋势完好（MA5/MA10向上）+ 无卖出信号 → 继续持有
- **减仓**：盈利超15% 或 出现量价背离 → 减仓50%锁定利润
- **卖出**：触发任一卖出信号 → 按信号要求执行
- **买入**：新的候选股触发买入信号 + 仓位未满 → 按建仓比例买入
