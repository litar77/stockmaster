# 进攻策略（牛市 + 热门）

## 策略定位

**市场环境**：牛市（Bull）+ 热门（Hot）

**核心思路**：牛市叠加交易活跃，是追求超额收益的最佳窗口期。策略重心在于捕捉龙头股的主升浪，通过快进快出最大化利润，同时严格止损保住本金。

**适用场景**：
- 大盘处于上升趋势，成交量持续放大
- 热点题材轮动活跃，龙头股连创新高
- 市场情绪亢奋，增量资金持续入场

## 仓位控制参数

| 参数 | 设置值 | 说明 |
|------|--------|------|
| 最大仓位比例 | 80% | 总持仓不超过账户80% |
| 单股最大比例 | 20% | 单只股票最大持仓不超过20% |
| 最大持股数量 | 5只 | 分散风险，同时管理能力有限 |
| 建仓节奏 | 50%+50% | 先建50%确认趋势后加满 |

| 参数 | 设置值 | 说明 |
|------|--------|------|
| 止损比例 | -5% | 亏损达到5%无条件清仓 |
| 止盈比例 | 15% | 盈利达到15%开始分批止盈 |
| 移动止损 | -5% | 从最高点回落5%触发清仓 |

## 选股方向

### 三大核心方向

**1. 龙头股（市场领导者）**
- 行业涨幅排名前列
- 成交量持续放大
- 股价创阶段新高或历史新高
- 具有板块号召力和资金凝聚力

**2. 热点题材（热门主题）**
- 政策利好或行业催化剂驱动
- 板块效应明显，资金集中涌入
- 题材新颖且具有想象力空间
- 关注新闻、政策、业绩预告等催化剂

**3. 强势股突破（技术突破）**
- 放量涨停突破前期高点或平台
- 突破时成交量为近期平均的1.5倍以上
- 突破后不回踩或快速回踩确认
- 均线系统呈多头排列

## 买入信号

当满足以下任一条件时，触发买入：

### 信号一：放量涨停突破前期高点
- 股价放量涨停，突破近20-60日高点
- 成交量较前一日放大1.5倍以上
- 涨停封板坚决，无开板或反复打开
- 次日高开（>2%）确认突破有效

**确认方式**：K线收盘价确认突破有效，次日开盘30分钟内验证强度

### 信号二：板块内龙头地位确认
- 个股在同一板块内涨幅领先
- 涨停时间早、封板力度强
- 带动同板块其他股票跟涨
- 获得短线资金集中认可

**确认方式**：通过板块排名和涨幅榜确认龙头地位

### 信号三：MACD金叉 + 成交量放大
- MACD指标在零轴上方或附近形成金叉（DIF从下往上穿越DEA）
- 金叉发生时成交量较5日均量放大30%以上
- 股价站上20日均线且均线向上发散
- 柱状体由绿转红且逐步放大

**确认方式**：MACD金叉当日收盘确认，次日量价配合验证

### 信号四：缠论买点（辅助参考）
- 日线级别出现第一类或第二类买点
- 底分型成立且停顿1-2天
- 次级别30分钟回调不破前低
- 配合放量中阳线确认

**确认方式**：底分型停顿后次日中阳线确认

## 卖出信号

当触发以下任一条件时，执行卖出：

### 信号一：触发止损价
- 亏损达到5%时无条件清仓
- 不抱侥幸心理，不等待反弹
- 严格执行，避免深套

### 信号二：盈利达到止盈目标
- 盈利15%以上开始分批止盈
- 先减仓50%锁定利润
- 剩余仓位使用移动止损

### 信号三：出现顶分型 + MACD柱收缩
- K线出现顶分型（高点降低）
- MACD红柱开始收缩或DIFF开始拐头向下
- 成交量与价格出现背离
- 预示短期调整或趋势反转

**确认方式**：顶分型停顿后确认，MACD柱收缩需连续2-3天观察

### 信号四：成交量异常放大但价格不涨
- 成交量突然放大至近期高点的1.5倍以上
- 股价涨幅却明显收窄或收跌
- 典型的出货信号
- 表明多方力量衰竭，空方开始主导

**确认方式**：当日K线形态确认，收长上影线或阴线更确认

## 风险管理提示

### 核心原则

1. **止损优先**：宁可小亏，不可大套。-5%止损是铁律，触达必须执行。

2. **仓位控制**：牛市虽好，但不要全仓。保留20%现金应对突发调整。

3. **分散持股**：同时持有3-5只股票，单只仓位不超过20%，避免黑天鹅事件。

4. **快进快出**：进攻策略持仓周期较短，3-7天未启动考虑换股。

5. **不追高**：追涨要在涨停前或涨停瞬间，已经涨停打开的不追。

### 情绪管理

- 不要试图抄底逃顶，趋势一旦形成有惯性
- 连续亏损3次后暂停交易，复盘后再战
- 盈利时不要骄傲，及时锁定利润才是真
- 关注大盘整体氛围，大盘转弱时降低仓位

### 资金管理

- 单次操作损失不超过总资金的2%
- 5%止损对应仓位为40%（2%/5%=40%）
- 保留部分资金做T+0或盘中灵活应对

## 操作时间窗口

| 时段 | 操作重点 |
|------|----------|
| 竞价（9:15-9:25） | 观察候选股竞价情况，高开超3%重点关注 |
| 早盘（9:30-10:30） | 最佳买入时段，龙头股通常在此期间封板 |
| 盘中（10:30-14:00） | 观察为主，寻找缩量回踩买入机会 |
| 尾盘（14:00-15:00） | 评估全天走势，决定是否加仓或减仓 |

## 持仓操作规则

| 情况 | 操作 | 条件 |
|------|------|------|
| 加仓 | 加至目标仓位 | 盈利中+回踩MA5/MA10获支撑+量能未萎缩 |
| 持仓 | 持有不动 | 趋势完好（MA5>MA10向上）+无卖出信号 |
| 减仓 | 减仓50% | 盈利超15%+出现量价背离 |
| 清仓 | 全部卖出 | 触发止损、移动止盈或卖点信号 |
| 买入 | 建仓50%+50% | 触发买入信号+仓位未满 |
