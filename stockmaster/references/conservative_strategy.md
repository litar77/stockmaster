# 保守观望策略（熊市 + 热门）

## 策略概述

### 市场定位

当市场处于**熊市（Bear）+ 热门（Hot）**双重叠加状态时，意味着整体趋势向下但市场交易依然活跃。这是A股市场中最危险的环境之一——下跌趋势中的高交易量往往制造反弹假象，诱惑投资者追高抄底，最终导致更大损失。

**核心原则：不赔钱就是胜利。**

在熊市热门市场中，策略的核心逻辑是：

- 严格控制仓位，将资金安全放在首位
- 只参与高确定性、高安全边际的短线机会
- 快速止损、快速止盈，不恋战
- 宁可错过机会，也要确保本金安全

### 策略收益预期

| 指标 | 目标值 |
|------|--------|
| 月度收益目标 | 0% ~ 3% |
| 最大单月亏损 | ≤ -5% |
| 胜率要求 | > 50% |
| 盈亏比 | ≥ 1.5:1 |

## 仓位控制参数

### 核心参数表

| 参数 | 数值 | 说明 |
|------|------|------|
| **最大持仓比例** | 30% | 总资金中用于股票的最大仓位 |
| **单票最大比例** | 10% | 单一股票最大占用资金比例 |
| **最大同时持股数** | 2只 | 同一时间持有的股票数量上限 |
| **最小现金比例** | 70% | 必须保持的最低现金储备 |

### 仓位管理细则

1. **建仓原则**：一律采用一次性建仓，不允许金字塔式加仓
2. **单票限制**：单只股票持仓不得超过总资金的10%
3. **总仓限制**：所有持仓市值不得超过总资金的30%
4. **分散原则**：同时持有不超过2只股票，每只股票平均分配仓位（约15%）
5. **动态调整**：若市场进一步恶化，仓位降至20%甚至更低

### 仓位执行示例

```
假设总资金：100万元

最大持仓市值：30万元
最大单票仓位：10万元
最小现金储备：70万元

同时持股：最多2只
每只股票建仓：约15万元（留10万元作为灵活备用）
```

## 止盈止损参数

### 核心参数表

| 参数 | 数值 | 说明 |
|------|------|
| **止损比例** | -3% | 亏损达到3%时无条件止损 |
| **止盈比例** | +5% | 盈利达到5%时锁定利润 |
| **移动止损** | -3% | 从最高点回落3%触发止盈 |
| **最大持有天数** | 5天 | 时间止损，超出必须清仓 |

### 止损机制详解

#### 固定止损（-3%）

- 当持仓亏损达到或超过3%时，必须在当日收盘前清仓
- 止损是自动执行的，不做任何讨论和犹豫
- 止损后若股票继续下跌，禁止抄底补仓

#### 移动止损（从最高点回落-3%）

- 记录持仓期间的最高成本价
- 当股价从最高点回落超过3%时，触发移动止盈
- 移动止盈一旦触发，立即清仓

#### 时间止损（5个交易日）

- 每只股票的持有时间不得超过5个交易日
- 无论盈亏状态，第5个交易日收盘前必须清仓
- 时间止损是铁律，不可违背

### 止盈机制详解

#### 固定止盈（+5%）

- 当持仓盈利达到或超过5%时，触发止盈
- 建议分批止盈：先卖出50%锁利，剩余持有但设置移动止损
- 移动止损设置在成本价上方1-2%位置

#### 止盈执行策略

```
盈利5%时的操作：
- 卖出50%持仓，锁定部分利润
- 剩余50%持仓设置移动止损（从最高点回落3%）
- 若股价继续上涨，不断上调移动止损位
- 第5个交易日无论如何清仓
```

## 选股方向

### 三大选股方向

在熊市热门市场中，选股必须严格遵循以下三个方向，拒绝一切模糊地带。

#### 方向一：超跌反弹（oversold bounce）

**适用场景**：某只股票因系统性风险或情绪恐慌导致大幅下跌，出现技术性反弹需求。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| 近20日跌幅 | > 20% | 充分下跌，空间到位 |
| RSI(14) | < 30 | 处于超卖区域 |
| 偏离MA20 | < -15% | 严重偏离均线 |
| 成交额 | > 5000万 | 保证流动性 |

**基本面要求**：

- 近一期财报盈利（PE为正）
- 无重大负面事件（业绩变脸、监管处罚、债务违约）
- 无减持/质押风险预警
- 所属行业非政策打压对象

**绝对排除**：

- ST、*ST股票
- 暴雷股、业绩造假股
- 连续跌停后首日开板股
- 上市不满60日的新股
- 市值低于50亿的股票

#### 方向二：短线热点快进快出（hot themes quick entry/exit）

**适用场景**：突发政策利好、突发事件刺激的短期热点板块。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| 板块效应 | 消息催化当日涨停 | 热点必须有硬逻辑 |
| 龙头股 | 板块中率先涨停且封板坚决 | 非龙头不参与 |
| 介入时机 | 消息首日或次日早盘 | 不追高位涨停 |
| 持仓周期 | 1-3天 | 热点轮动快，不可恋战 |

**操作要点**：

- 只做消息首日的强势涨停股
- 次日高开超过5%不追入
- 若开板后快速回封可考虑介入
- 持有不超过3个交易日

**绝对排除**：

- 跟风涨停的杂毛股
- 封板不坚决的股票
- 热点已经扩散2日以上的板块
- 消息已经被市场充分预期的热点

#### 方向三：绝对安全边际品种（value with margin of safety）

**适用场景**：具有明确估值支撑的优质股票，被市场错杀。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| PE | < 行业平均PE的70% | 估值明显低于行业 |
| PB | < 2 | 破净或低市净率 |
| 股息率 | > 3% | 有现金分红支撑 |
| 近三年净利润 | 均为正且稳定 | 业绩稳定可靠 |
| 行业地位 | 细分行业龙头 | 竞争壁垒高 |

**基本面要求**：

- 主营业务清晰，无多元恶化风险
- 现金流充裕，无债务危机
- 管理层稳定，无诚信问题
- 行业前景稳定，非夕阳产业

## 买入信号条件

### 三大买入信号

只有同时满足以下条件时，才考虑买入建仓。任何单一条件都不足以触发买入。

#### 信号一：连续下跌后缩量企稳

**技术特征**：

- 连续下跌3-5个交易日，跌幅超过10%
- 第4-6个交易日成交量萎缩至前期均量的30-50%
- 股价不再创新低，在某一级别的均线附近获得支撑
- K线形态出现小阳线、十字星等企稳信号

**确认要点**：

```
✅ 连续下跌：下跌过程中无反弹
✅ 缩量企稳：量能萎缩至地量水平
✅ 支撑明显：在MA5/MA10/MA20某一位置企稳
✅ 阳线确认：出现止跌阳线或十字星
```

#### 信号二：底分型 + RSI超卖

**底分型定义**（缠论基础）：

- 连续三根K线，中间一根的低点是三根K线中最低的
- 第三根K线的高点高于第一根K线的高点
- 底分型出现后等待确认——次日K线收盘价高于底分型最高点

**RSI超卖标准**：

- RSI(14) < 30：标准超卖
- RSI(14) < 25：深度超卖，更强信号
- RSI(6) < 20：短期超卖辅助确认

**复合信号确认**：

```
✅ 底分型成立：次根K线收盘价突破底分型最高点
✅ RSI超卖：RSI(14) < 30，至少满足一项
✅ 量能配合：底分型当日或次日成交量放大
✅ 趋势确认：股价站稳5日均线
```

#### 信号三：突发利好刺激的短线机会

**利好类型优先级**：

| 优先级 | 利好类型 | 参与价值 |
|--------|----------|----------|
| P0 | 国家级政策利好（降准、重大行业政策） | 高 |
| P1 | 部委级政策利好 | 中高 |
| P2 | 地方政府或企业重大合同 | 中 |
| P3 | 行业性利好传闻（未证实） | 低 |

**参与条件**：

```
✅ 必须是P0或P1级别的硬逻辑利好
✅ 利好发布当日个股涨停或高开3%以上
✅ 封板坚决，换手率合理（3-10%）
✅ 板块内多只股票联动涨停
✅ 介入时机：利好当日或次日早盘30分钟内
```

### 买入前必查清单

每次买入前，必须逐项确认以下7个条件：

| 序号 | 检查项 | 标准 |
|------|--------|------|
| 1 | 大盘环境 | 指数处于低位，非加速下跌中 |
| 2 | 止损位确定 | 买入前明确止损价，亏损不超过3% |
| 3 | 仓位合规 | 买入后总仓位 ≤ 30%，单票 ≤ 10% |
| 4 | 持有周期 | 计划持有1-5天，不超过5个交易日 |
| 5 | 盈亏比 | 预期盈利空间 ≥ 止损空间的1.5倍 |
| 6 | 流动性 | 日成交额 > 5000万，可正常买卖 |
| 7 | 基本面 | 无重大利空，业绩稳定 |

## 卖出信号条件

### 五大卖出信号

触发以下任一条件，必须立即执行卖出操作。

#### 信号一：触发止损（-3%）

**触发条件**：持仓亏损达到或超过3%

**执行动作**：

- 立即在当日收盘前清仓
- 不做任何等待和侥幸
- 止损后禁止当日再次买入同一股票

**示例**：

```
买入价：10.00元
止损价：9.70元（-3%）
触发止损：股价下跌至9.70元或以下
执行：立即卖出全部持仓
```

#### 信号二：盈利达到目标（+5%）

**触发条件**：持仓盈利达到或超过5%

**执行动作**：

- 建议先卖出50%持仓，锁定部分利润
- 剩余50%持仓设置移动止损（从最高点回落3%）
- 保留的仓位若继续上涨，不断上调止损位

**操作策略**：

```
盈利5%时的分级操作：
├─ 立即卖出50% → 锁定2.5%利润
├─ 剩余50%设置移动止损 → 从最高点回落3%止盈
└─ 第5个交易日无论如何清仓
```

#### 信号三：持有超过5个交易日

**触发条件**：从买入当日起算，持有满5个交易日

**执行动作**：

- 第5个交易日收盘前清仓
- 无论盈亏、无论何种理由
- 这是铁律，不可打破

**日历计算**：

```
T日买入 → T+1、T+2、T+3、T+4、T+5 → 第5个交易日
示例：周一买入 → 周五收盘前必须清仓
```

#### 信号四：反弹遇阻明显

**技术特征**：

- 放量上影线：当日成交量放大但股价冲高回落
- 长上影线：上影线长度超过实体部分的2倍
- 遇阻位置：反弹至MA5/MA10/MA20附近受阻
- 阴包阳：次日K线吞没前日阳线

**识别要点**：

```
✅ 放量上影线：当日成交额是前5日均量的1.5倍以上
✅ 冲高回落：收盘价低于当日均价
✅ 均线压制：股价在MA5/MA10/MA20下方运行
✅ 抛压明显：盘口出现大单砸盘迹象
```

#### 信号五：移动止损触发

**触发条件**：股价从持仓期间最高点回落超过3%

**执行动作**：

- 立即清仓
- 不等待反弹
- 防止利润大幅回吐

## 时间止损规则

### 5日时间止损铁律

时间止损是熊市热门市场中最重要的一道防线。

**规则定义**：从买入当日起，所有持仓必须在5个交易日内了结。

**规则逻辑**：

```
短线机会的本质是"超跌反弹"或"消息刺激"
这类机会的时间窗口通常只有3-5个交易日
超过5个交易日，利好兑现或反弹动力衰竭
继续持有只会增加不确定性风险
```

### 时间止损执行细则

| 持有天数 | 操作策略 |
|----------|----------|
| T日（买入） | 确认建仓，设置止损价 |
| T+1~T+3 | 观察走势，若盈利超5%可止盈50% |
| T+4 | 若仍持仓，做好次日清仓准备 |
| T+5 | 无条件清仓，无论盈亏 |

### 时间止损的特殊情况

**提前触发卖出信号**：若在5日内触发其他卖出信号（止损/止盈/遇阻），按触发信号执行，不受时间限制。

**盈利情况下**：若持仓已盈利，优先使用移动止损或固定止盈退出，不必须等到第5日。

**亏损情况下**：若持仓亏损，止损或第5日清仓，以更早触发者为准。

## 风险管理要点

### 资金保存优先原则

在熊市热门市场中，**不亏钱就是赢**。本策略的核心目标是保护本金，等待更好的市场机会。

### 五大风险管理原则

#### 原则一：不亏大钱

```
宁可错过10次机会，也不亏1次大的
单次最大亏损控制在总资金的3%以内
月度最大亏损控制在总资金的5%以内
```

#### 原则二：不补仓

```
亏损持仓绝对不允许补仓
补仓是熊市中最危险的操作
摊薄成本只会放大亏损
```

#### 原则三：不追高

```
绝对不在股价上涨超过5%后追高买入
错过机会好过追高被套
熊市追涨是最大风险来源
```

#### 原则四：快速认错

```
买入后股价未按预期上涨，立即检查逻辑
买入逻辑被破坏，无条件止损
不要给亏损持仓"多一天"观察时间
```

#### 原则五：保持现金流

```
始终保持70%以上的现金储备
现金是熊市中最宝贵的资产
手中有现金，才有捡便宜的机会
```

### 风险监控指标

| 指标 | 警戒线 | 处理措施 |
|------|--------|----------|
| 单月亏损 | -3% | 降低仓位至20%以下 |
| 单月亏损 | -5% | 暂停新建仓，只处理现有持仓 |
| 单日最大回撤 | -2% | 检查所有持仓，准备减仓 |
| 连续亏损次数 | 3次 | 暂停操作3个交易日复盘 |

## 持仓操作指令规则

当系统策略判定为"保守观望"时，对现有持仓股的指令规则：

| 指令 | 条件 | 操作 |
|------|------|------|
| **加仓** | ❌ 禁止 | 此策略下绝对不加仓 |
| **持仓** | 盈利中 + 未触发卖出信号 + 持有天数<5 | 可继续持有 |
| **减仓** | 盈利超5% | 减仓50%，剩余设移动止盈 |
| **清仓** | 触发任一卖出信号 或 持有超5日 | 立即清仓 |
| **买入** | 极少机会 + 严格满足全部买入条件 + 仓位未满 | 不超过10%仓位 |
