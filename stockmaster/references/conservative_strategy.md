# 保守观望策略（熊市 + 热门）

## 策略概述

### 市场定位

当市场处于**熊市（Bear）+ 热门（Hot）**双重叠加状态时，意味着整体趋势向下但市场交易依然活跃。这是A股市场中最危险的环境之一——下跌趋势中的高交易量往往制造反弹假象，诱惑投资者追高抄底，最终导致更大损失。

**核心原则：不赔钱就是胜利。**

在熊市热门市场中，策略的核心逻辑是：

- 严格控制仓位，将资金安全放在首位
- 只参与高确定性、高安全边际的短线机会
- 快速止损、快速止盈，不恋战
- 宁可错过机会，也要确保本金安全

### 策略收益预期

| 指标 | 目标值 |
|------|--------|
| 月度收益目标 | 0% ~ 3% |
| 最大单月亏损 | ≤ -5% |
| 胜率要求 | > 50% |
| 盈亏比 | ≥ 1.5:1 |

## 仓位控制参数

### 核心参数表

| 参数 | 数值 | 说明 |
|------|------|------|
| **最大持仓比例** | 30% | 总资金中用于股票的最大仓位 |
| **单票最大比例** | 10% | 单一股票最大占用资金比例 |
| **最大同时持股数** | 2只 | 同一时间持有的股票数量上限 |
| **最小现金比例** | 70% | 必须保持的最低现金储备 |

### 仓位管理细则

1. **建仓原则**：一律采用一次性建仓，不允许金字塔式加仓
2. **单票限制**：单只股票持仓不得超过总资金的10%
3. **总仓限制**：所有持仓市值不得超过总资金的30%
4. **分散原则**：同时持有不超过2只股票，每只股票平均分配仓位（约15%）
5. **动态调整**：若市场进一步恶化，仓位降至20%甚至更低

### 仓位执行示例

```
假设总资金：100万元

最大持仓市值：30万元
最大单票仓位：10万元
最小现金储备：70万元

同时持股：最多2只
每只股票建仓：约15万元（留10万元作为灵活备用）
```

## 止盈止损参数

### 核心参数表

| 参数 | 数值 | 说明 |
|------|------|------|
| **止损比例** | -3% | 亏损达到3%时无条件止损 |
| **止盈比例** | +5% | 盈利达到5%时锁定利润 |
| **移动止损** | -3% | 从最高点回落3%触发止盈 |
| **最大持有天数** | 5天 | 时间止损，超出必须清仓 |

### 止损机制详解

#### 固定止损（-3%）

- 当持仓亏损达到或超过3%时，必须在当日收盘前清仓
- 止损是自动执行的，不做任何讨论和犹豫
- 止损后若股票继续下跌，禁止抄底补仓

#### 移动止损（从最高点回落-3%）

- 记录持仓期间的最高成本价
- 当股价从最高点回落超过3%时，触发移动止盈
- 移动止盈一旦触发，立即清仓

#### 时间止损（5个交易日）

- 每只股票的持有时间不得超过5个交易日
- 无论盈亏状态，第5个交易日收盘前必须清仓
- 时间止损是铁律，不可违背

### 止盈机制详解

#### 固定止盈（+5%）

- 当持仓盈利达到或超过5%时，触发止盈
- 建议分批止盈：先卖出50%锁利，剩余持有但设置移动止损
- 移动止损设置在成本价上方1-2%位置

#### 止盈执行策略

```
盈利5%时的操作：
- 卖出50%持仓，锁定部分利润
- 剩余50%持仓设置移动止损（从最高点回落3%）
- 若股价继续上涨，不断上调移动止损位
- 第5个交易日无论如何清仓
```

## 选股模型：五级漏斗筛选体系

本策略采用**A股熊市标的严选模型**，通过五级漏斗层层筛选，从全市场5000+标的中识别出具备**"安全边际"与"反弹潜力"**的极少数优质标的。

### 模型核心理念

- **风控第一**：熊市不言底，唯有信号确认才纳入视野（右侧思维）
- **确定性优先**：上涨概率权重 > 涨幅空间权重
- **技术为王**：融合缠论背驰、量价关系、情绪周期识别高质量标的

---

### 第一级：宏观环境识别（定筛选阈值）

**输入数据**：
- 大盘指数（上证/创业板）日K线、5日均线数据
- 全市场成交量数据、涨跌停家数统计

**筛选算法与判断逻辑**：

#### 均线趋势判定

```python
# 判定指数与5日线的位置关系
if 指数收盘价 > 5日均线:
    环境状态 = "右侧企稳区"
    环境系数 = 1.0
    筛选门槛 = 标准
else:
    环境状态 = "左侧弱势区"
    环境系数 = 0.7
    筛选门槛 = 提高（要求更严格的底背离结构）
```

**逻辑说明**：若指数在5日线下方，判定为"左侧弱势区"，模型自动提高后续筛选门槛；若指数站上5日线，判定为"右侧企稳区"，维持标准门槛。

#### 量能逆转信号

```python
# 检测是否出现"缩量后的标志性放量"
近期成交量 = 近5日平均成交量
前期均量 = 近20日平均成交量

if 近期成交量 < 前期均量 * 0.6:  # 缩量至60%以下
    量能状态 = "地量"
    if 当日成交量 > 近期成交量 * 1.5:  # 放量突破
        量能信号 = "逆转确认"
```

**逻辑说明**：地量之后见地价，唯有成交量趋势逆转（缩量转放量），才视为有效筛选窗口。

**输出结果**：
- 环境系数（0.5-1.0）：用于调整后续评分权重
- 有效标的池：全市场A股与ETF列表

### 第二级：板块方向定位（定筛选范围）

**输入数据**：
- 上一级输出的标的列表
- 申万行业指数、热门概念板块指数涨幅排名
- 板块内涨停股数量、板块资金流向

**筛选算法与判断逻辑**：

#### 热点与赚钱效应识别

```python
# 筛选条件
板块涨幅 > 1.5% 且 板块内涨停股 >= 2只

# 逻辑说明
热点板块 = []
for 板块 in 所有板块:
    if 板块.涨幅 > 1.5% and 板块.涨停股数量 >= 2:
        热点板块.append(板块)
```

**逻辑说明**：熊市无全面行情，唯有热点板块具备合力。筛选板块涨幅>1.5%且板块内涨停股≥2只的板块。

#### 新题材优先原则

```python
# 新题材识别
新题材 = 近3日内首次出现的题材概念
老热点 = 近期有过"赚钱效应"的板块（如前期龙头股所在板块）

if 新题材存在:
    优先板块 = 新题材板块
else:
    优先板块 = 近期有赚钱效应的老热点板块
```

**逻辑说明**：若有新题材涌现，优先标记新题材板块；若无新题材，则筛选近期有过"赚钱效应"的老热点板块。

#### 防御性板块识别（熊市特有）

```python
# 防御性板块定义
防御板块 = ["医药生物", "公用事业", "银行", "食品饮料"]

if 市场环境系数 < 0.6:  # 市场环境极差
    备选池 = 防御板块
```

**逻辑说明**：若市场环境极差，自动筛选医药、高股息、公用事业等防御性板块作为备选池。

**输出结果**：
- 核心板块池（Top 3-5个板块）
- 数据流向：提取核心板块内的所有成分股及相关ETF

# 个股流动性筛选

```python
# 风险剔除条件
风险条件 = [
    "ST状态",

    "近期巨额解禁",

if 近3个月有过涨停 and 涨停次日无闷杀:
    股性活跃 = True
```

**逻辑说明**：筛选近期（近3个月）有过涨停记录，且涨停次日无"闷杀"（大幅低开低走）记录的个股。股性活跃意味着资金关注度高。

**输出结果**：

- ETF初选池（约3-5只）

### 第四级：缠论与技术深度评分（定标的成色）

**输入数据**：
- 初选池标的的日K线、30分钟K线、分时图、成交量数据

**筛选算法与判断逻辑**：

#### 缠论结构识别（核心算法）

**底背离扫描**：

```python
def 检测底背离(股价, MACD绿柱):
    """
    检测日线或30分钟级别是否存在底背离结构
    底背离：股价新低，但MACD绿柱缩短/黄白线金叉
    """
    近期低点 = 近20日最低价
    前期低点 = 前20-40日最低价
    
    if 近期低点 < 前期低点:  # 股价创新低
        近期MACD = MACD绿柱[近期对应位置]
        前期MACD = MACD绿柱[前期对应位置]
        
        if 近期MACD > 前期MACD:  # MACD绿柱缩短
            return "底背离确立"
    
    return "无底背离"
```

**判定**：出现底背离，标记为"高安全边际"。

**中枢突破扫描**：

```python
def 检测中枢突破(股价, 成交量):
    """
    检测是否放量突破前期下跌中枢，且回踩不破中枢上沿（类三买）
    """
    前期中枢上沿 = 近40-60日价格密集区上沿
    突破确认 = 收盘价 > 前期中枢上沿 * 1.03
    放量确认 = 成交量 > 20日均量 * 1.5
    回踩不破 = 最低价 > 前期中枢上沿
    
    if 突破确认 and 放量确认 and 回踩不破:
        return "中枢突破确认"
    
    return "无突破"
```

**判定**：出现突破，标记为"趋势启动"。

#### 量价梯度分析

**梯量放大**：

```python
# 筛选成交量呈现 V1 < V2 < V3（温和放大）且股价重心上移的标的
成交量序列 = 近3日成交量
价格序列 = 近3日收盘价

if 成交量序列[0] < 成交量序列[1] < 成交量序列[2]:
    if 价格序列[0] < 价格序列[1] < 价格序列[2]:
        梯量放大 = True
```

**缩量回调**：

```python
# 对于趋势股，筛选缩量回踩关键均线（如10日/20日线）的标的
if 股价 > 10日均线 and 股价 < 5日均线:  # 回踩10日线
    if 成交量 < 5日均量 * 0.8:  # 缩量
        缩量回调 = True
```

#### 龙头地位确认

```python
def 确认龙头地位(个股, 板块):
    """
    在板块内筛选涨幅最大、涨停最早、带动效应最强的个股
    """
    板块内个股 = 板块.成分股
    
    涨幅排名 = sorted(板块内个股, key=lambda x: x.涨幅, reverse=True)
    涨停时间排名 = sorted(板块内个股, key=lambda x: x.涨停时间)
    
    if 个股 in 涨幅排名[:3] and 个股 in 涨停时间排名[:3]:
        return "板块龙头"
    
    return "非龙头"
```

**逻辑说明**：在板块内筛选涨幅最大、涨停最早、带动效应最强的个股，标记为"龙头"。

#### 技术评分计算

```python
def 计算技术评分(标的):
    """
    技术评分（0-100分）
    - 缠论结构（40%）
    - 量价配合（30%）
    - 龙头地位（30%）
    """
    缠论得分 = 0
    if 底背离确立:
        缠论得分 = 40
    elif 中枢突破确认:
        缠论得分 = 35
    
    量价得分 = 0
    if 梯量放大:
        量价得分 = 30
    elif 缩量回调:
        量价得分 = 20
    
    龙头得分 = 0
    if 板块龙头:
        龙头得分 = 30
    
    总分 = 缠论得分 + 量价得分 + 龙头得分
    return min(总分, 100)
```

**输出结果**：
- 技术评分（0-100分）
- 数据流向：评分排名前5的标的进入最终输出

**筛选算法与判断逻辑**：

#### 上涨概率计算

```python
def 计算上涨概率(技术评分, 板块热度, 环境系数):
    """
    P = 基准概率(60%) + (技术评分 - 70) × 0.5% + 板块热度加成
    """
    基准概率 = 60.0
    技术加成 = (技术评分 - 70) * 0.5
    板块加成 = 板块热度 * 5  # 板块热度0-5分
    
    上涨概率 = (基准概率 + 技术加成 + 板块加成) * 环境系数
    return min(max(上涨概率, 30), 95)  # 限制在30%-95%
```

#### 涨幅空间预测

```python
def 预测涨幅空间(标的):
    """
    基于上方压力位距离、筹码峰分布计算潜在涨幅区间
    """
    上方压力位 = 计算压力位(标的)
    当前价格 = 标的.收盘价
    
    最大涨幅 = (上方压力位 - 当前价格) / 当前价格 * 100
    保守涨幅 = 最大涨幅 * 0.6  # 取60%作为保守估计
    
    return (保守涨幅, 最大涨幅)
```

## 选股方向

### 三大选股方向

在熊市热门市场中，选股必须严格遵循以下三个方向，拒绝一切模糊地带。

#### 方向一：超跌反弹（oversold bounce）

**适用场景**：某只股票因系统性风险或情绪恐慌导致大幅下跌，出现技术性反弹需求。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| 近20日跌幅 | > 20% | 充分下跌，空间到位 |
| RSI(14) | < 30 | 处于超卖区域 |
| 偏离MA20 | < -15% | 严重偏离均线 |
| 成交额 | > 5000万 | 保证流动性 |

**基本面要求**：

- 近一期财报盈利（PE为正）
- 无重大负面事件（业绩变脸、监管处罚、债务违约）
- 无减持/质押风险预警
- 所属行业非政策打压对象

**绝对排除**：

- ST、*ST股票
- 暴雷股、业绩造假股
- 连续跌停后首日开板股
- 上市不满60日的新股
- 市值低于50亿的股票

#### 方向二：短线热点快进快出（hot themes quick entry/exit）

**适用场景**：突发政策利好、突发事件刺激的短期热点板块。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| 板块效应 | 消息催化当日涨停 | 热点必须有硬逻辑 |
| 龙头股 | 板块中率先涨停且封板坚决 | 非龙头不参与 |
| 介入时机 | 消息首日或次日早盘 | 不追高位涨停 |
| 持仓周期 | 1-3天 | 热点轮动快，不可恋战 |

**操作要点**：

- 只做消息首日的强势涨停股
- 次日高开超过5%不追入
- 若开板后快速回封可考虑介入
- 持有不超过3个交易日

**绝对排除**：

- 跟风涨停的杂毛股
- 封板不坚决的股票
- 热点已经扩散2日以上的板块
- 消息已经被市场充分预期的热点

#### 方向三：绝对安全边际品种（value with margin of safety）

**适用场景**：具有明确估值支撑的优质股票，被市场错杀。

**筛选标准**：

| 指标 | 要求 | 说明 |
|------|------|------|
| PE | < 行业平均PE的70% | 估值明显低于行业 |
| PB | < 2 | 破净或低市净率 |
| 股息率 | > 3% | 有现金分红支撑 |
| 近三年净利润 | 均为正且稳定 | 业绩稳定可靠 |
| 行业地位 | 细分行业龙头 | 竞争壁垒高 |

**基本面要求**：

- 主营业务清晰，无多元恶化风险
- 现金流充裕，无债务危机
- 管理层稳定，无诚信问题
- 行业前景稳定，非夕阳产业

## 买入信号条件

### 三大买入信号

只有同时满足以下条件时，才考虑买入建仓。任何单一条件都不足以触发买入。

#### 信号一：连续下跌后缩量企稳

**技术特征**：

- 连续下跌3-5个交易日，跌幅超过10%
- 第4-6个交易日成交量萎缩至前期均量的30-50%
- 股价不再创新低，在某一级别的均线附近获得支撑
- K线形态出现小阳线、十字星等企稳信号

**确认要点**：

```
✅ 连续下跌：下跌过程中无反弹
✅ 缩量企稳：量能萎缩至地量水平
✅ 支撑明显：在MA5/MA10/MA20某一位置企稳
✅ 阳线确认：出现止跌阳线或十字星
```

#### 信号二：底分型 + RSI超卖

**底分型定义**（缠论基础）：

- 连续三根K线，中间一根的低点是三根K线中最低的
- 第三根K线的高点高于第一根K线的高点
- 底分型出现后等待确认——次日K线收盘价高于底分型最高点

**RSI超卖标准**：

- RSI(14) < 30：标准超卖
- RSI(14) < 25：深度超卖，更强信号
- RSI(6) < 20：短期超卖辅助确认

**复合信号确认**：

```
✅ 底分型成立：次根K线收盘价突破底分型最高点
✅ RSI超卖：RSI(14) < 30，至少满足一项
✅ 量能配合：底分型当日或次日成交量放大
✅ 趋势确认：股价站稳5日均线
```

#### 信号三：突发利好刺激的短线机会

**利好类型优先级**：

| 优先级 | 利好类型 | 参与价值 |
|--------|----------|----------|
| P0 | 国家级政策利好（降准、重大行业政策） | 高 |
| P1 | 部委级政策利好 | 中高 |
| P2 | 地方政府或企业重大合同 | 中 |
| P3 | 行业性利好传闻（未证实） | 低 |

**参与条件**：

```
✅ 必须是P0或P1级别的硬逻辑利好
✅ 利好发布当日个股涨停或高开3%以上
✅ 封板坚决，换手率合理（3-10%）
✅ 板块内多只股票联动涨停
✅ 介入时机：利好当日或次日早盘30分钟内
```

### 买入前必查清单

每次买入前，必须逐项确认以下7个条件：

| 序号 | 检查项 | 标准 |
|------|--------|------|
| 1 | 大盘环境 | 指数处于低位，非加速下跌中 |
| 2 | 止损位确定 | 买入前明确止损价，亏损不超过3% |
| 3 | 仓位合规 | 买入后总仓位 ≤ 30%，单票 ≤ 10% |
| 4 | 持有周期 | 计划持有1-5天，不超过5个交易日 |
| 5 | 盈亏比 | 预期盈利空间 ≥ 止损空间的1.5倍 |
| 6 | 流动性 | 日成交额 > 5000万，可正常买卖 |
| 7 | 基本面 | 无重大利空，业绩稳定 |

## 卖出信号条件

### 五大卖出信号

触发以下任一条件，必须立即执行卖出操作。

#### 信号一：触发止损（-3%）

**触发条件**：持仓亏损达到或超过3%

**执行动作**：

- 立即在当日收盘前清仓
- 不做任何等待和侥幸
- 止损后禁止当日再次买入同一股票

**示例**：

```
买入价：10.00元
止损价：9.70元（-3%）
触发止损：股价下跌至9.70元或以下
执行：立即卖出全部持仓
```

#### 信号二：盈利达到目标（+5%）

**触发条件**：持仓盈利达到或超过5%

**执行动作**：

- 建议先卖出50%持仓，锁定部分利润
- 剩余50%持仓设置移动止损（从最高点回落3%）
- 保留的仓位若继续上涨，不断上调止损位

**操作策略**：

```
盈利5%时的分级操作：
├─ 立即卖出50% → 锁定2.5%利润
├─ 剩余50%设置移动止损 → 从最高点回落3%止盈
└─ 第5个交易日无论如何清仓
```

#### 信号三：持有超过5个交易日

**触发条件**：从买入当日起算，持有满5个交易日

**执行动作**：

- 第5个交易日收盘前清仓
- 无论盈亏、无论何种理由
- 这是铁律，不可打破

**日历计算**：

```
T日买入 → T+1、T+2、T+3、T+4、T+5 → 第5个交易日
示例：周一买入 → 周五收盘前必须清仓
```

#### 信号四：反弹遇阻明显

**技术特征**：

- 放量上影线：当日成交量放大但股价冲高回落
- 长上影线：上影线长度超过实体部分的2倍
- 遇阻位置：反弹至MA5/MA10/MA20附近受阻
- 阴包阳：次日K线吞没前日阳线

**识别要点**：

```
✅ 放量上影线：当日成交额是前5日均量的1.5倍以上
✅ 冲高回落：收盘价低于当日均价
✅ 均线压制：股价在MA5/MA10/MA20下方运行
✅ 抛压明显：盘口出现大单砸盘迹象
```

#### 信号五：移动止损触发

**触发条件**：股价从持仓期间最高点回落超过3%

**执行动作**：

- 立即清仓
- 不等待反弹
- 防止利润大幅回吐

## 时间止损规则

### 5日时间止损铁律

时间止损是熊市热门市场中最重要的一道防线。

**规则定义**：从买入当日起，所有持仓必须在5个交易日内了结。

**规则逻辑**：

```
短线机会的本质是"超跌反弹"或"消息刺激"
这类机会的时间窗口通常只有3-5个交易日
超过5个交易日，利好兑现或反弹动力衰竭
继续持有只会增加不确定性风险
```

### 时间止损执行细则

| 持有天数 | 操作策略 |
|----------|----------|
| T日（买入） | 确认建仓，设置止损价 |
| T+1~T+3 | 观察走势，若盈利超5%可止盈50% |
| T+4 | 若仍持仓，做好次日清仓准备 |
| T+5 | 无条件清仓，无论盈亏 |

### 时间止损的特殊情况

**提前触发卖出信号**：若在5日内触发其他卖出信号（止损/止盈/遇阻），按触发信号执行，不受时间限制。

**盈利情况下**：若持仓已盈利，优先使用移动止损或固定止盈退出，不必须等到第5日。

**亏损情况下**：若持仓亏损，止损或第5日清仓，以更早触发者为准。

## 风险管理要点

### 资金保存优先原则

在熊市热门市场中，**不亏钱就是赢**。本策略的核心目标是保护本金，等待更好的市场机会。

### 五大风险管理原则

#### 原则一：不亏大钱

```
宁可错过10次机会，也不亏1次大的
单次最大亏损控制在总资金的3%以内
月度最大亏损控制在总资金的5%以内
```

#### 原则二：不补仓

```
亏损持仓绝对不允许补仓
补仓是熊市中最危险的操作
摊薄成本只会放大亏损
```

#### 原则三：不追高

```
绝对不在股价上涨超过5%后追高买入
错过机会好过追高被套
熊市追涨是最大风险来源
```

#### 原则四：快速认错

```
买入后股价未按预期上涨，立即检查逻辑
买入逻辑被破坏，无条件止损
不要给亏损持仓"多一天"观察时间
```

#### 原则五：保持现金流

```
始终保持70%以上的现金储备
现金是熊市中最宝贵的资产
手中有现金，才有捡便宜的机会
```

### 风险监控指标

| 指标 | 警戒线 | 处理措施 |
|------|--------|----------|
| 单月亏损 | -3% | 降低仓位至20%以下 |
| 单月亏损 | -5% | 暂停新建仓，只处理现有持仓 |
| 单日最大回撤 | -2% | 检查所有持仓，准备减仓 |
| 连续亏损次数 | 3次 | 暂停操作3个交易日复盘 |

## 模型核心特点总结

1. **纯客观筛选**：模型仅负责从海量数据中识别符合"缠论背驰"、"量价配合"、"板块热点"特征的标的，不包含任何主观的交易指令。

2. **熊市适应性**：引入"明王心法"的5日线判定与流动性筛选，专门规避熊市常见的阴跌与流动性枯竭风险。

3. **技术与基本面共振**：将游资的"龙头战法"、"热点跟随"与缠论的"结构分析"结合，既有短线的爆发力，又有技术的安全边际。

4. **五级漏斗体系**：通过层层递进的筛选机制，从5000+标的中精准定位极少数优质标的，通过率低至0.08%，确保高确定性。

5. **量化评分机制**：采用技术评分（缠论40%+量价30%+龙头30%）和概率计算，实现客观、可量化的标的评估。
