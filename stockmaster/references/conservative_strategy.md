# 模块4：保守观望策略（熊市 + 热市）

## 策略定位

市场整体下行但交易仍然活跃，通常出现在熊市中的反弹阶段或恐慌性抛售期。这是最危险的市场状态之一——高活跃度容易让人产生反弹幻觉。核心思路：**不赔钱就是胜利**。

## 选股条件

此策略下不主动选股建仓。如果必须交易（如现有持仓需要换股），严格按以下条件：

### 超跌反弹候选（仅限短线）

**筛选条件：**
- 近20日跌幅 > 20%（充分下跌）
- 基本面无重大利空（非暴雷股）
- 市值 > 50亿（避免小盘股流动性风险）
- 日成交量 > 5000万（保证流动性）

**技术确认：**
- RSI(14) < 25（超卖区域）
- 出现底分型（缠论确认）
- 成交量较前一日放大1.5倍以上（资金介入信号）
- 偏离MA20超过 -15%（超跌幅度足够）

**缠论辅助判断：**
- 仅在30分钟或60分钟级别出现第一类买点时考虑
- 日线级别必须有背驰信号（下跌力度衰竭）
- 不做没有背驰信号的反弹

### 绝对排除

- 任何没有盈利的公司
- 近期有减持/质押预警的股票
- 行业整体处于政策打压的板块
- 连续跌停后首日开板的股票（风险极高）

## 买入信号（极其严格）

| 信号 | 条件 | 确认方式 |
|------|------|----------|
| 超跌反弹 | RSI<25 + 底分型 + 放量 | 必须三个条件同时满足 |
| 政策底 | 重大利好政策出台 + 指数级别放量阳线 | 等待次日确认不回落 |

**买入前必须确认：**
1. 有明确的止损位（买入前就确定）
2. 仓位不超过总资金的10%
3. 预期持有时间不超过5个交易日

## 卖出信号（快速严格）

| 信号 | 条件 | 操作 |
|------|------|------|
| 快速止盈 | 盈利达到 5%-8% | 立即清仓，不贪 |
| 严格止损 | 亏损达到 -3% | 无条件清仓 |
| 时间止损 | 持有超过5个交易日未盈利 | 清仓不等 |
| 反弹结束 | 反弹至MA10/MA20附近受阻回落 | 清仓 |
| 缠论卖点 | 30分钟级别出现卖点 | 清仓 |

## 仓位管理

| 参数 | 设置 |
|------|------|
| 总仓位上限 | 30% |
| 单票仓位上限 | 10% |
| 同时持股数量 | 最多2只 |
| 建仓方式 | 一次性建仓，不加仓 |
| 现金比例 | 至少保持70%现金 |

## 操作纪律

- **绝不追涨**：错过就错过，熊市中追涨是最大风险
- **绝不补仓**：亏损的持仓不补仓，只止损
- **快进快出**：持股周期不超过5个交易日
- **宁可踏空**：错过10次机会也不亏一次大的

## 持仓操作指令规则

当策略为"保守观望"时，对持仓股的操作判断：

- **加仓**：❌ 此策略下不加仓任何股票
- **持仓**：盈利中且未触发卖出信号 + 持有天数<5 → 可继续持有
- **减仓**：盈利超5% → 减仓50%或清仓锁定利润
- **卖出**：触发任一卖出信号 或 持有超5日 → 立即清仓
- **买入**：极少数超跌反弹机会 + 严格满足所有买入条件 → 不超过10%仓位
