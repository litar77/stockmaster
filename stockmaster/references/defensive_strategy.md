# 模块5：空仓关注策略（熊市 + 冷市）

## 策略定位

市场下行且交易冷清，通常是熊市末期或深度调整期。这是最安全也最无聊的阶段。核心思路：**保存实力，空仓等待，关注底部信号**。

## 操作原则

**此策略下不进行任何主动交易。** 唯一的任务是：
1. 清掉现有持仓（如果还有的话）
2. 每日监控底部信号
3. 维护观察股票池，为下一轮行情做准备

## 清仓规则

如果进入此策略时仍有持仓：

| 持仓状态 | 操作 |
|----------|------|
| 盈利中 | 逢高分批卖出，3个交易日内清完 |
| 小幅亏损（<5%） | 直接清仓 |
| 较大亏损（5%-15%） | 等反弹至-5%以内后清仓，最多等5个交易日 |
| 深度套牢（>15%） | 评估基本面，基本面好的可保留观察，基本面差的止损清仓 |

## 底部信号监测

每日运行以下检查，当多个信号同时出现时，提示可能的底部区域：

### 量能信号
- 全市场日成交量降至近一年最低30%分位以下
- 连续5日以上缩量，量能萎缩至高峰期的1/3以下
- 地量出现（单日成交量为近60日最低）

### 估值信号
- 上证指数PE（TTM）接近历史低位（<12倍）
- 破净股数量占比 > 10%
- 全市场股息率中位数 > 3%

### 情绪信号
- 涨停家数连续多日 < 20
- 跌停家数明显减少（恐慌消退）
- 新开户数大幅下降（散户离场）

### 政策信号
- 降准/降息
- 印花税下调
- 国家队入场（汇金/社保增持ETF）
- 重大维稳表态

### 缠论信号
- 周线级别出现背驰（下跌力度衰竭）
- 月线级别接近前期中枢区间下沿
- 日线级别出现第一类买点的雏形

## 观察股票池管理

即使空仓，也应维护一个观察池（10-20只），为转势后第一时间建仓做准备：

**入池条件：**
- 行业龙头或细分领域隐形冠军
- 基本面优秀（ROE>15%，连续3年营收增长>10%）
- 跌幅已充分（从高点回撤>30%）
- 估值合理（PE处于近5年低位区域）
- 机构持仓稳定或逆市增持

**出池条件：**
- 基本面出现恶化
- 出现重大利空（财务造假、监管处罚等）
- 行业逻辑发生根本性变化

## 仓位管理

| 参数 | 设置 |
|------|------|
| 总仓位目标 | 0%（完全空仓） |
| 允许的例外 | 无 |
| 现金配置 | 100%现金，可配置货币基金或逆回购 |

## 转策略信号

当以下条件满足时，考虑从空仓切换到其他策略：

- 底部信号中有3个以上同时出现 → 可小仓位试探（切换到保守观望）
- 行情趋势判断连续3日评为"牛" → 切换到对应的牛市策略
- 重大政策利好 + 大盘放量上涨 → 关注但不急于入场

## 持仓操作指令规则

当策略为"空仓关注"时，对持仓股的操作判断：

- **加仓**：❌ 绝对不加仓
- **持仓**：❌ 不应有持仓，如有则按清仓规则处理
- **减仓**：如有持仓 → 按清仓规则逐步减至零
- **卖出**：如有持仓 → 按清仓规则执行
- **买入**：❌ 绝对不买入
