# 空仓关注策略（Defensive Cash Position Strategy）

**市场条件：熊市(Bear) + 冷门(Cold)**

---

## 一、策略概述

当市场处于熊市且交易冷清时，这意味着市场处于明显的下降趋势中，赚钱效应极差。此阶段的首要任务是**清仓离场、保存实力**，而不是试图在跌势中寻找机会。

### 策略核心思想

- **零持仓**：在确认市场企稳之前，不持有任何股票
- **主动关注**：将精力放在观察和研究上，寻找先于大盘企稳的标的
- **耐心等待**：等待至少2-3个底部信号同时出现后才考虑入场
- **宁可错过，不可做错**：在明显的下降趋势中，现金为王

## 二、仓位控制

| 参数 | 值 | 说明 |
|------|-----|------|
| **max_position_ratio** | **0%** | 最大持仓比例，清仓状态 |
| 正常仓位 | 0% | 不持有任何股票 |
| 试探仓位 | 0% | 绝对不进行试探性建仓 |
| 加仓权限 | 禁止 | 任何情况下都禁止加仓 |

### 执行原则

1. **立即清仓**：策略确认后，在第一个开盘机会清空所有持仓
2. **不复利思维**：熊市冷门阶段，不要想着用小仓位"试试水"
3. **保留弹药**：账户资金是未来反攻的唯一资本，不可随意消耗

---

## 三、核心原则

### 3.1 资本保全原则

```
熊市冷门阶段 >>> 资本保全 > 任何收益机会
```

- 10万现金在熊市后 = 10万购买力 + 抄底机会
- 10万持仓在熊市后 = 6万亏损 + 深套动弹不得

### 3.2 机会成本原则

熊市冷门阶段 **不操作的成本远低于错误操作的成本**：

| 操作 | 潜在成本 |
|------|----------|
| 持有现金观望 | 踏空风险（但可快速纠错） |
| 错误抄底被套 | 本金永久损失风险 |
| 频繁止损 | 资金持续消耗 |

### 3.3 逆向思维原则

- 当所有人都在讨论"抄底"时，往往还没到真正的底部
- 真正的底部信号是**沉默的**，没有大量"抄底"声音
- 冷门市场是筛选优质标的的最佳时期，因为有充足的时间研究

## 四、底部信号识别

在考虑入场之前，必须观察到以下**至少2个以上**的底部信号同时出现。单一信号不足以确认底部。

### 4.1 MACD底背离（MACD Bullish Divergence）

```
识别方法：
- 价格创出新低
- MACD指标未创新低（或金叉位置明显抬升）
- 背离次数越多，信号越强
```

**判断标准**：
- 一次背离：可能是反弹信号，可观察但不可行动
- 二次背离：底部概率大增，重点关注
- 三次背离：强烈底部信号，等待入场条件确认

### 4.2 成交量连续缩量（Continuously Declining Volume）

```
识别方法：
- 连续10个交易日以上成交量递减
- 成交量萎缩至前期高点的30%以下
- 出现地量（近半年最低量）
```

**判断标准**：
- 量能萎缩代表卖压衰竭
- 地量之后可能见地价
- 但需注意：缩量可能持续很长时间，不可贸然入场

### 4.3 放量企稳（Volume Spike with Stabilization）

```
识别方法：
- 单日或连续2-3日成交量放大至均量2倍以上
- 价格不再创新低，开始企稳
- 阳线数量多于阴线
```

**判断标准**：
- 放量是资金介入的信号
- 必须配合价格企稳，单日放量下跌不算
- 放量后缩量整理的时间越长，后续上涨概率越大

### 4.4 底部放量（Bottoming Volume）

```
识别方法：
- 在相对低位（前期跌幅30%以上）出现明显放量
- 放量时以阳线为主
- 后续回调时成交量快速萎缩（主力建仓特征）
```

**判断标准**：
- 底部放量70%以上概率是主力建仓信号
- 需观察后续缩量回调是否缩至放量前的水平
- 放量位置是否是重要支撑位

### 4.5 多次测试前低不破（Testing Lows Without Breaking）

```
识别方法：
- 价格连续多次下探前一轮低点
- 但每次都能获得支撑反弹
- 反弹低点逐步抬高
```

**判断标准**：
- 3次以上测试不破：强支撑信号
- 每次反弹高点高于前一次：上升趋势萌芽
- 配合缩量：更加确认

## 五、入场条件（当底部确认后）

**警告**：只有在底部信号确认后，才能考虑以下入场条件。入场条件必须**同时满足**。

### 5.1 必须同时满足的条件

| 条件 | 具体要求 | 达标 |
|------|----------|------|
| 信号数量 | 至少2个以上底部信号同时出现 | ☐ |
| 缩量整理时间 | 缩量整理时间超过10个交易日 | ☐ |
| 放量阳线 | 放量阳线突破5日均线 | ☐ |

### 5.2 入场时机选择

```
第一步：信号确认
├── MACD底背离出现（二次以上更佳）
├── 成交量萎缩至地量水平
└── 价格不再创新低

第二步：等待整理
└── 缩量整理超过10个交易日

第三步：突破确认
├── 放量阳线出现
├── 阳线实体突破5日均线
└── 当日涨幅大于3%

第四步：分批入场
└── 突破确认后首个交易日入场30%
```

### 5.3 入场仓位管理

即使满足所有入场条件，首次入场仓位不超过计划仓位的**30%**：

| 阶段 | 仓位 | 条件 |
|------|------|------|
| 首次入场 | 30% | 放量阳线突破5日均线 |
| 回调不破入场 | +20% | 回调至5日均线附近缩量企稳 |
| 趋势确认入场 | +30% | 突破20日均线，量能持续放大 |
| 保留现金 | 20% | 永远保留20%现金作为机动 |

## 六、观望期行动指南

在市场未出现明确底部信号之前，保持观望但**不无所事事**。

### 6.1 每日观察清单

```
每日必做：
├── 记录大盘成交量变化（缩量还是放量）
├── 记录大盘指数位置（是否创新低）
├── 关注涨跌家数比（上涨家数/下跌家数）
├── 对比沪深300与创业板指强弱
└── 记录自选股异动情况

禁止行为：
├── ✗ 根据"感觉跌够了"买入
├── ✗ 买入跌幅较大的股票"摊薄成本"
├── ✗ 买入"便宜"的股票
├── ✗ 听信"内部消息"买入
└── ✗ 因为无聊而随便买卖
```

### 6.2 自选股跟踪

**选择自选股的标准**：
- 行业龙头，基本面优质
- 前期跌幅超过40%（超跌）
- 近期成交量开始温和放大
- 有明显的主力资金介入痕迹

**跟踪记录表**：

| 日期 | 股票 | 收盘价 | 成交量 | 5日均线 | 10日均线 | 20日均线 | 异动备注 |
|------|------|--------|--------|---------|----------|----------|----------|
|      |      |        |        |         |          |          |          |

### 6.3 关注先于大盘企稳的标的

**重点观察信号**：
- 大盘仍在下跌，但某股票已不再创新低
- 大盘缩量时，该股开始温和放量
- 该股提前于大盘开始反弹

**意义**：这类股票往往是下一轮行情的领头羊

### 6.4 基本面研究

利用熊市冷门阶段的时间优势，深入研究：

```
研究维度：
├── 行业地位：是否行业龙头/细分龙头
├── 财务状况：近3年营收/利润增长率
├── 现金流：经营现金流是否为正
├── 估值水平：PE/PB历史分位点
├── 股东结构：机构持仓变化
└── 行业前景：政策支持/市场需求

估值合理标准：
- PE处于历史20%分位以下
- 或预期当年业绩增长30%以上
```

## 七、风险管理

### 7.1 核心风险控制

```
风险管理优先级：
1. 不亏损 > 少亏损 > 保本
2. 宁可踏空，不可被套
3. 一次深度套牢需要多次盈利才能弥补
```

### 7.2 亏损容忍

| 阶段 | 最大亏损容忍 | 处理方式 |
|------|-------------|----------|
| 首次建仓30%后 | -5% | 止损出局，等待下一次机会 |
| 加仓20%后 | 累计-8% | 全部止损 |
| 趋势确认后 | 累计-12% | 无条件止损 |

### 7.3 止损原则

```
止损纪律：
- 买入后亏损超过5%，立即止损
- 不做任何"等反弹再卖"的幻想
- 止损后30个交易日内不得再次买入同一标的
- 连续止损3次，强制休息10个交易日
```

### 7.4 仓位动态调整

```
市场环境变化时的仓位调整：

熊市+冷门 → 清仓（0%持仓）
熊市+转暖 → 观察信号，不盲目加仓
底部信号确认 → 30%试探仓
趋势确认 → 逐步加仓至80%
市场过热 → 逐步减仓至50%
```
