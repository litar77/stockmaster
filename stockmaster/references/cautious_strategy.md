# 模块3：谨慎乐观策略（牛市 + 冷市）

## 策略定位

市场大趋势向上但交易冷清，通常出现在牛市的调整阶段或启动初期。核心思路：**低风险保守盈利，耐心等待市场回暖**。

## 选股条件

### 主要筛选标准

**第一轮：基本面筛选**
- PE（TTM）< 行业中位数（估值不贵）
- ROE（近四季度）> 12%（盈利能力强）
- 营收同比增长 > 10%（成长性）
- 资产负债率 < 60%（财务健康）

**第二轮：技术面确认**
- 股价在MA60和MA120上方（中长期趋势完好）
- MA60向上或走平（非下行趋势）
- 近20日振幅 < 15%（波动可控）
- 成交量缩至20日均量的0.6-0.9倍（缩量整理，非恐慌出逃）

**第三轮：价值特征**
- 股息率 > 1.5%（有分红回报）
- 市值 > 100亿（蓝筹白马特征）
- 机构持仓比例 > 10%（有机构认可）

**第四轮：缠论确认**
- 日线级别处于中枢震荡或中枢上移阶段
- 周线级别趋势向上（至少有一个完成的上涨中枢）
- 未出现顶分型（日线级别）

### 排除条件

- ST、*ST 股票
- 近30日有高管减持的股票
- 近一季度业绩预亏的股票
- 所属行业处于下行周期的股票

## 买入信号

| 信号 | 条件 | 确认方式 |
|------|------|----------|
| 缩量回调支撑 | 回调至MA60/MA120附近，缩量企稳 | 连续2日不再创新低 |
| 底部放量 | 缩量调整后出现单日量能放大1.5倍+收阳 | 次日继续收阳确认 |
| 业绩催化 | 业绩预增/回购公告/利好消息 + 技术面支撑 | 公告日+次日量价配合 |
| 缠论第二类买点 | 回调不破前低，在中枢上沿获支撑 | 底分型成立 |

## 卖出信号

| 信号 | 条件 | 操作 |
|------|------|------|
| 目标止盈 | 盈利达到 10%-15% | 减仓50% |
| 趋势破位 | 跌破MA60且3日内不收回 | 清仓 |
| 固定止损 | 亏损达到 -8% | 无条件清仓 |
| 基本面恶化 | 业绩低于预期 / 重大利空 | 评估后决定 |
| 缠论卖点 | 日线出现第一类卖点 | 减仓50% |
| 市场转熊 | 行情状态切换为熊市 | 逐步减仓至策略仓位 |

## 仓位管理

| 参数 | 设置 |
|------|------|
| 总仓位上限 | 60% |
| 单票仓位上限 | 15% |
| 同时持股数量 | 3-6只 |
| 建仓方式 | 分3批建仓（30%→30%→40%），每批间隔≥2个交易日 |
| 预留资金 | 至少保留40%现金应对市场变化 |

## 操作节奏

- 以中线持股为主，持股周期 2-4周
- 不追涨，重点关注回调到位的机会
- 每周评估一次持仓，非必要不做日内操作
- 密切关注市场温度变化，如转热则切换至进攻策略

## 持仓操作指令规则

当策略为"谨慎乐观"时，对持仓股的操作判断：

- **加仓**：回调至支撑位且基本面无变化 + 当前仓位未达目标 → 按分批建仓计划加仓
- **持仓**：趋势完好（在MA60上方）+ 基本面良好 → 继续持有
- **减仓**：盈利超10% 或 接近前期压力位 → 减仓50%锁定部分利润
- **卖出**：跌破MA60 或 基本面出现问题 → 清仓止损
- **买入**：新股票满足筛选条件 + 仓位未满 + 出现买入信号 → 首批30%建仓
